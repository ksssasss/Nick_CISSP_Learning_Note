# What is a Strategy? How?
1. 構劃願景( Vision )：「可見的」美好未來
2. 設定目標 ( Goats/ Objectives)
3. 制定戰略( Strategy )
	- 下達政策( Policy )、指示
4. 戰略執行

Example: CISO perspetive
建立資安制度
	找顧問
	找CB驗證
